# Global Warming Potentials

GWP values for different greenhouse gases and time horizons based on
IPCC guidelines.

## Format

A data frame with GWP values:

- Gas:

  Greenhouse gas name

- GWP_20:

  20-year global warming potential

- GWP_100:

  100-year global warming potential (most commonly used)

- GWP_500:

  500-year global warming potential

## Source

IPCC Assessment Reports
