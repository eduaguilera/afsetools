# Aggregate Primary Products to CBS Items

Aggregates primary production footprints from production items to CBS
(commodity balance sheet) items.

## Usage

``` r
Agg_primary(df)
```

## Arguments

- df:

  Primary product footprint data frame

## Value

A data frame with footprints aggregated to CBS item level

## Examples

``` r
if (FALSE) { # \dontrun{
primary_cbs <- Agg_primary(primary_footprints)
} # }
```
