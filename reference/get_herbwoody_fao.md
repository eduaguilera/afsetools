# Get Herbaceous and Woody Land from FAO

Extracts herbaceous (arable) and woody (permanent crops) land area from
FAO land use data.

## Usage

``` r
get_herbwoody_fao()
```

## Value

A data frame with columns: Year, area, Herb_Woody, Land_tot

## Examples

``` r
if (FALSE) { # \dontrun{
land_data <- get_herbwoody_fao()
} # }
```
