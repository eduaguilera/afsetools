# Biological Nitrogen Fixation Parameters

Parameters for calculating biological nitrogen fixation by crop type,
including symbiotic and non-symbiotic fixation.

## Format

A data frame with BNF coefficients:

- Name_BNF:

  Crop or system name

- Ndfa:

  Proportion of nitrogen derived from atmosphere (0-1)

- Leguminous_share:

  Fraction of leguminous species (0-1)

- BGN:

  Below-ground nitrogen parameter

- NHI:

  Nitrogen harvest index
