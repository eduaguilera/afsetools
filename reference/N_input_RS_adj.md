# N Input Root:Shoot Adjustment Factors

Adjustment factors for root-to-shoot ratios based on nitrogen input
levels. Higher N availability reduces root allocation (functional
equilibrium theory).

## Format

A data frame with columns:

- N_input_class:

  N application category name.

- N_input_min:

  Lower bound of N rate (kg N/ha/yr).

- N_input_max:

  Upper bound of N rate (kg N/ha/yr).

- RS_adjustment:

  Multiplicative factor on default RS ratio.

- Source:

  Literature source reference.

## Source

Poorter & Nagel (2000) New Phytologist 147:135-147.
