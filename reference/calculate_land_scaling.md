# Calculate Land Scaling Factors

Calculates scaling factors to account for cropping intensity (fallow,
double cropping, etc.).

## Usage

``` r
calculate_land_scaling()
```

## Value

A data frame with columns: Year, area, Herb_Woody, land_scaling

## Examples

``` r
if (FALSE) { # \dontrun{
scaling_factors <- calculate_land_scaling()
} # }
```
