# IPCC Root Biomass Coefficients

Root-to-shoot ratios and reference root biomass values for estimating
below-ground crop biomass, with adjustments for N input and irrigation.

## Format

A data frame with columns:

- IPCC_crop:

  IPCC crop category name.

- RS_default:

  Default root-to-shoot ratio.

- RS_low_N:

  RS ratio under low N input (\<60 kg N/ha).

- RS_high_N:

  RS ratio under high N input (\>120 kg N/ha).

- RS_irrigated:

  RS ratio under irrigation.

- RS_rainfed:

  RS ratio under rainfed conditions.

- BG_ref_MgDMha:

  Reference below-ground biomass (Mg DM / ha).

- Source:

  Literature source reference.

## Source

IPCC (2019) 2019 Refinement, Vol.4, Ch.11. Bolinder et al. (2007).
Poorter & Nagel (2000). Benjamin et al. (2014).
