# Aggregate Processed Products from Countries to Origins

Aggregates processed product footprints from country-level to
origin-level.

## Usage

``` r
Agg_processed(df)
```

## Arguments

- df:

  Processed product footprint data frame

## Value

A data frame with processed footprints aggregated by origin

## Examples

``` r
if (FALSE) { # \dontrun{
processed_agg <- Agg_processed(processed_footprints)
} # }
```
