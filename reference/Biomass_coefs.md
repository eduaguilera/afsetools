# Biomass Coefficients

Main coefficient table containing dry matter, nitrogen, carbon, and
energy content for different biomass types. Used for converting between
different biomass units.

## Format

A data frame with coefficients for biomass conversion including:

- item:

  Item code or name

- DM_content:

  Dry matter content (fraction)

- N_content:

  Nitrogen content (kg N / kg DM)

- C_content:

  Carbon content (kg C / kg DM)

- energy_MJ_kg:

  Energy content (MJ / kg)

## Source

Harmonized from multiple scientific databases and literature
