# Calculate Embodied Impact of Processed Products

Traces impacts through processing chains using processing shares and
conversion coefficients.

## Usage

``` r
Calc_impact_processed(df)
```

## Arguments

- df:

  Product footprint data frame

## Value

A data frame with processed product footprints

## Examples

``` r
if (FALSE) { # \dontrun{
processed_fp <- Calc_impact_processed(primary_footprints)
} # }
```
