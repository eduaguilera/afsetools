Package Functions

This R package provides tools for calculating environmental footprints in agro-food systems.

Installation

devtools::install_github("eduaguilera/afsetools")

Quick Start

library(afsetools)
load_general_data()  # Load all data objects
?calculate_footprints  # Get help for functions
